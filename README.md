# myPython
바이너리 루프에 오신 것을 환영합니다.
이 프로그램은 이진법의 편리한 계산과 확인을 목적으로 설계되었습니다.
최초 숫자는 0부터 시작하며,공백을 입력하면 다음 숫자로 넘어갑니다.
+나 -를 입력하여 현재 값에서 입력된 숫자만큼 더하거나 뺍니다.
또는 그냥 숫자만 입력하여 그 숫자에 대한 계산값을 볼 수 있습니다.

Welcome to Binary Loop.
This program is designed for convenient calculations and verification of binary numbers.
The initial number starts at 0, and entering a space advances to the next number.
Enter + or - to add or subtract the entered number from the current value.
Or, simply enter a number to see the calculated value.
