# myPython
안녕하세요. 제 코드로 많은 도움이 되셨으면 좋겠습니다.
이 프로그램은 이진수의 편리한 계산과 확인을 목적으로 설계되었습니다.
최초 숫자는 0부터 시작하며,공백을 입력하면 다음 숫자로 넘어갑니다.
'+' ,'-', '*', '/' 를 입력하여 간단한 사칙연산을 수행할 수 있습니다.
또는 숫자만 입력하여 그 숫자로 넘어갑니다.
U를 입력하여 특정 문자에 대한 유니코드 이진수를 확인하거나, 유니코드 코드 포인트의 이진수 번역을 할 수 있습니다.

This program is designed to facilitate binary calculations and verification.
The initial number starts at 0, and entering a space advances to the next number.
You can perform simple arithmetic operations by entering '+', '-', '*', and '/'.
Or, you can simply enter a number to advance to that number.
You can type U to see the Unicode binary for a specific character, or to get the binary translation of a Unicode code point.
